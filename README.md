# SoftStackersAI Pomodoro Timer

A lightweight, single-file Pomodoro timer app built with HTML, CSS, and JavaScript.  
Includes:
- Custom work/break/long-break durations
- Auto-start option
- Session log with chart
- Theme toggle (light/dark)
- Embedded SoftStackersAI branding (feel free to modify)
- Red tomato favicon
- Works locally in any modern browser

## Usage

1. **Clone or download** this repository.
2. Open the HTML file in Chrome:
   - Double-click the `.html` file  
   - Or drag it into a browser window
3. Start your Pomodoro session and track work/break cycles.
